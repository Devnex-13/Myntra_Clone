# Myntra Clone Frontend

This folder contains the React + Vite frontend for the Myntra clone application.

## Install

```bash
cd 3-myntra-clone
npm install
```

## Run in development

```bash
npm run dev
```

## Build for production

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```

## Linting

```bash
npm run lint
```

## Notes

- The frontend uses React 19 with Vite, Redux Toolkit, React Router, Bootstrap, and React Icons.
- It is configured as an ES module project (`type: module`).
- The frontend expects the backend API to be available on port `8080`.
